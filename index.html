Spatial Goniometry Cloud Platform
프로젝트 내용
• SG Cloud Platform을 활용하여 실내 공간을 파악하여 수집한 데이터를 오버레이한 후 지도의 형태로 표현 가능
• Base map, Space Resolution map, Object map 등과 같이 지도를 여러 레이어로 제작함으로써 사용자의 요구사항 및 환경에 적합한 지도를 생성해 제공할 수 있음
• SG Cloud S/W System을 활용해 생성된 실내 지도와 실외 지도 및 GPS의 연동 가능하므로 실외에서도 건물 내부의 특정 장소를 목적지로 설정할 수 있고, 실내 오브젝트의 실시간 정보를 받아 사용자에게 제공 가능
• 기존에 존재하는 실내 지도와 SG Cloud S/W System 내에 저장된 Object map data를 연동하여 실제 사용 중인 실내 공간을 유동적으로 인지 가능
• 실내 공간의 오브젝트를 인식해서 코드화 및 분류 하여 지도에 표현했기 때문에 사용자가 찾고자 하는 장소를 카테고리 별로 간편하게 탐색이 가능
• KOREN 환경 내 SG Cloud S/W System에서 사용자에게 지도를 전송할 때 수신 받는 디바이스 환경 및 사용자의 요구사항을 수집해서 맞춤형 품질 지도를 제공할 수 있음
• 별도의 프로그램의 설치가 필요하지 않고 웹 페이지를 통해 지도 측위 정보를 제공해 플랫폼의 제한이 없기 때문에 사용자의 접근이 용이함

Ant Car
프로젝트 내용
• Hands Free Auto Following 기능으로 ANT Car가 사용자를 인식하고 목적지로 설정하여 자동 주행
• 군집성 주행을 위해 여러 대의 ANT Car간에 통신을 하여 실시간 주행 환경 데이터를 공유
• Emergency Mode로 돌발 상황에 빠르게 대처할 수 있는 시스템을 위해 LIDAR 및 각종 센서와 Visioning Library를 활용
• 어플리케이션을 개발하여 사용자가 ANT Car의 시작/종료/긴급조종 모드를 사용할 수 있도록 도와줌
